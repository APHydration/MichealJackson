# GitHub Upload Folder

This folder contains the clean set of files for the Weekly Creator Payout Portal.

Safe to upload:
- app
- components
- lib
- supabase
- package.json
- package-lock.json
- tsconfig.json
- next.config.ts
- next-env.d.ts
- middleware.ts
- vercel.json
- .gitignore
- .env.example
- README.md

Do not upload secrets from `.env.local` into GitHub.
Add real environment variable values in Vercel instead.
